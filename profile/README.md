# iFinance — Portable Simple Income & Expense Tracker

## About iFinance

iFinance provides a complete personal finance tracking environment for users who want simple income and expense management. iFinance combines transaction logging, budgeting, reporting, charts, and bank import. iFinance enables users to track spending, set savings goals, and understand financial habits. iFinance runs on desktop platforms. iFinance is trusted by individuals, students, families, and freelancers worldwide.

## Deep Dive — Personal Finance Notes

Transaction tracking: log income and expenses, categories, tags, notes
Budgeting: set spending limits by category, track progress with charts
Bank import: import bank statements (CSV, QIF, OFX), automatic categorization
Reporting: spending by category, income vs expenses, cash flow
Visualization: pie charts, bar charts, trends over time, calendar view
